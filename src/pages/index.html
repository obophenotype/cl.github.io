The Cell Ontology

The Cell Ontology (CL) is an OBO Foundry ontology covering the domain of canonical, natural biological cell types.

FAIR principles

The Cell Ontology is built on FAIR principles and interoperates well with other ontologies. It works in concert with the multi-species anatomy ontology UBERON, and is used as the reference ontology for cell types in the Gene ontology (GO).

Active Community

The Cell Ontology actively engages the community, editors from multiple projects embeded in the team, active and responsive discussions in the issue tracker, and constant improvements both in content and technical development of the ontology.

Ontology Tools Integrated

The Cell Ontology is also integrated into tools used by the community such as ubergraph, and is uploaded into popular ontology browsers such as the Ontology Lookup Service (OLS) and Ontobee.

Widely Compatible

The cell ontology is released in multiple forms to be compatible with releases done in owl, obo, and json format. It also comes in multiple variants including full (all imports merged in, and classified using a reasoner), base (only axioms belonging to the ontology), and simple (a subset of the ontology useful for tool development), all of which are openly accessed in our GitHub page
